TreesTreesTrees: This project creates a fractal tree that appears on the canvas when you run the code. The setup function is called once when the sketch starts, and draw is called repeatedly, which allows for the animation of the tree growing. The branch function is called recursively to create the fractal pattern. The len parameter controls the length of the branch, and the angle variable controls the angle of rotation for the branches. You can experiment with different values for these variables to change the appearance of the fractal.
